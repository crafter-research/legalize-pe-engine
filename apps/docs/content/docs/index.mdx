**legalize-pe** es la legislación peruana como datos abiertos y consultables por
agentes: un corpus de 21k+ normas (nacional + 26 jurisdicciones regionales,
cada reforma un commit de git), más un servicio hosted que responde preguntas
legales **citando la fuente oficial**.

Tres principios: **open corpus, reproducible pipeline, inspectable answer.** Esto
no es asesoría legal.
